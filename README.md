# CoalOS

![Alt Text](https://github.com/Downtime1147/CoalOS/blob/main/Dependencies/C__Windows_py.exe%202021-04-03%2015-42-41.gif)

Welcome, and thank you for taking the time to check out CoalOS !


CoalOS is a hacking simulator made entierly in Python where you can (hopefully) enjoy the wonders of breaching the Data Protection Act
from the comfort of your own home, withought getting arrested and sent to jail!

If you need help, use the 'help' command or read the Help.txt file for more instructions on how to use it.

Notice : The issue with some graphics not displaying correctly on some resolutions is currently being addressed and should be fixed in the next update.
